Порядок восстановления исключенных участников
Эколого-Экономического региона "Алтай"

1. Восстановление хозяйствующих субъектов в качестве участников ЭЭР "Алтай" осуществляется Дирекцией.
2. Для восстановления экономического субъекта в качестве участника ЭЭР "Алтай" он обязан погасить задолженность по обязательным платежам в бюджет и внебюджетные фонды и взносам в Дирекцию, а также до истечения квартала, в котором произошло исключение, предоставить:
2.1. заявление о восстановлении в качестве участника ЭЭР "Алтай" по форме разрабатываемой Дирекцией;
2.2. информационные письма из ГНИ и внебюджетных фондов об отсутствии задолженности по обязательным платежам в бюджет и внебюджетные фонды;
3. Восстановление в качестве участника ЭЭР "Алтай" осуществляется посредством отмены решения Дирекции об исключении экономического субъекта из состава участников ЭЭР "Алтай".
4. Статус участника ЭЭР "Алтай" и связанные с ним льготы предоставляются только с нового квартала, следующего за кварталом, в котором произошло исключение экономического субъекта из состава участников ЭЭР "Алтай".
5. Восстановление хозяйствующих субъектов в качестве участников ЭЭР "Алтай" может быть произведено только один раз.
